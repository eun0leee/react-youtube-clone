## 1. 담당 기능 구현

### 1.1 헤더/사이드바
  
- 헤더 검색바 반응형
- 데스크탑 사이즈에서는 버튼 클릭시, 토글로 열림
- 태블릿과 모바일 사이즈에서는 버튼 클릭시, 모달로 열림
  - 모달일 때
    - 사이드바 외 영역 클릭시, 모달 닫기
    - 버튼으로 모달 닫기
- 스크롤은 hover 시에만 노출

### 1.2 레이아웃
  
- 영상 디테일 페이지 : 사이드바 모달 형태
- 그외 페이지
  - 사이드바 반응형에 따라, main 영역 사이즈 변경됨
- 스크롤 디자인
